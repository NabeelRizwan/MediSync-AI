# Usage Guide

## Installation
1. Get Google Gemini API key from https://ai.google.dev
2. Add to Kaggle Secrets as GOOGLE_API_KEY
3. Run the notebook

## Usage

### Option 1: Interactive Mode
- System prompts for patient data
- Enter: demographics, vitals, medications, history, symptoms
- Get: diagnosis and treatment plan

### Option 2: Demo Mode
- Pre-loaded test cases
- Instant results

### Option 3: Batch Mode
- Process 3+ cases at once
- Get batch results with metrics

## Medical Image Analysis
- Upload JPG or PNG image
- Get automatic clinical interpretation

## Output Format
- Research findings
- Differential diagnosis with probabilities
- Complete treatment plan with medications and dosages
- Monitoring schedule
- Patient education
