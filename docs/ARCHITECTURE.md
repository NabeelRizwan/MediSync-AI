# System Architecture

## Overview
MediSync AI implements a multi-agent orchestration pattern.

## Components
- Research Agent (Gemini)
- Diagnostic Agent (Gemini)
- Treatment Agent (Gemini)
- Visual Agent (Gemini)

## Data Flow
Patient Data → Research → Diagnosis → Treatment → Output
